# Evidence Protection System Using Blockchain

A blockchain-based digital evidence management and chain-of-custody system designed to securely store, verify, and manage digital evidence files using Ethereum smart contracts and decentralized storage.

---

## Project Overview

This project provides a secure platform for managing digital evidence while maintaining integrity, transparency, and tamper resistance using blockchain technology.

The system enables:
- Secure evidence uploads
- Blockchain-based evidence verification
- Chain-of-custody tracking
- User authentication and authorization
- IPFS-based decentralized file storage
- Evidence access management

---

## Features

- Upload and manage digital evidence
- Immutable blockchain logging
- Evidence integrity verification
- Chain-of-custody monitoring
- User authentication system
- Admin dashboard
- Smart contract integration
- IPFS decentralized storage support
- React-based frontend interface
- Node.js backend APIs
